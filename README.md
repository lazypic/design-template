#### SVG to PNG
- macOS 에서는 Inkscape를 이용해서 svg파일을 png로 바꿀 수 있습니다.

```bash
$ /Applications/Inkscape.app/Contents/Resources/bin/inkscape -z /Users/woong/design-template/logo_1080p.svg -e /Users/woong/design-template/logo_1080p.png
```
